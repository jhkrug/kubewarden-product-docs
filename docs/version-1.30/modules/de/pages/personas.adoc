## {project-name}-Personas

[cols="1,2", options="header"]
|===
|Persona |Beschreibung

|**Richtlinienbenutzer**
|Jemand, der eine Richtlinie nimmt und sie in einem Cluster verwendet. Sie stellen Richtlinien bereit und beobachten die Ergebnisse. Sie konfigurieren die Richtlinieneinstellungen, schreiben aber nicht den internen Code der Richtlinie (falls vorhanden).

|**Betreiber**
|Jemand, der {short-project-name} in einem Cluster betreibt. Installiert und wartet {short-project-name}.

|**Richtlinienverteiler**
|Jemand, der eine Richtlinie geschrieben hat und sie mit anderen teilen möchte. Sie möchten, dass die Richtlinie von der Rolle „Richtlinienbenutzer“ konsumiert wird. Ein Verteiler von {short-project-name}-Richtlinien an Orte wie ArtifactHub oder private OCI-Repositories.

|**Richtlinienentwickler**
|Eine Person, die Richtlinien für {short-project-name} entwirft und schreibt, damit andere sie konsumieren können.

|**Integrator**
|Baut mit, unter Verwendung von oder auf {short-project-name} auf. Vielleicht eine benutzerdefinierte Benutzeroberfläche oder ein Hilfsprogramm zum Generieren von Richtlinien.

|**Entwickler**
|Eine Person, die am {short-project-name}-Produktsatz arbeitet. Baut {short-project-name} selbst.
|===
